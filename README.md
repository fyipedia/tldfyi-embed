# tldfyi-embed

Embed TLDFYI widgets on any website. Part of the [FYIPedia](https://fyipedia.com) developer tools ecosystem.

**Coming soon.** Visit [TLDFYI](https://tldfyi.com) for more information.

## Embed Widget

Embed [TLDFYI](https://tldfyi.com) widgets on any website with [tldfyi-embed](https://widget.tldfyi.com):

```html
<script src="https://cdn.jsdelivr.net/npm/tldfyi-embed@1/dist/embed.min.js"></script>
<div data-tldfyi="entity" data-slug="example"></div>
```

Zero dependencies · Shadow DOM · 4 themes (light/dark/sepia/auto) · [Widget docs](https://widget.tldfyi.com)
